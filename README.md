### Installation process :

- clone repo
- navigate to project folder
- install all dependencies with npm
- run with npm


```bash
$ git clone https://github.com/RandolphG/Time-Sync-Too-Electron-React-.git
$ cd project
$ npm i
$ npm run electron
```

- electron
